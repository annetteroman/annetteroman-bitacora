Hablan los botones

Proyecto

Lima, 2022 Annette R.

INTEGRANTES
- Giannella.
- Estrella.
- Annette.

1. Introducción
Teclado de botones adaptable para niños con problemas musculares. Un sencillo teclado USB de 5 botones que utiliza Logitech Adaptive Gaming Kit para niños con afecciones musculares como miopatía o distrofia muscular.

2. Caracterisitcas
- El teclado puede escribir hasta 20 letras.
- Puede formar prácticamente cualquier palabra.
- Principalmenete un sistema de comunicación hablado y no de escritura.
- Se ah eliminado todas las letras redundantes y las hemos ordenado por frecuencia de uso.
Ejemplo: 'b' y 'v' se convierten simplemente en 'b', o 'i' e 'y' se convierten en 'i'. Dado que texto a voz leerá la palabra "muy" casi igual que "mui". También eliminamos la 'h' ya que esta letra es muda.
- Pero creo que este sistema es perfectamente adaptable a la ideosincrasia de muchos idiomas.

3. Tarjeta de comunicación

¿Como funciona?

- El sistema de botones funciona como un sistema de coordenadas con esta tarjeta de letras
- Espera dos pulsaciones de teclas diferentes para poder escribir una letra.
Ejemplo:
- Así, para escribir la letra 'T', apretaremos el botón 3 (ya que la letra está en la fila 3) y el botón 2 (ya que la letra está en la columna 2)
- Para escribir la letra 'Z', presione el botón 4, el sistema esperará una segunda presión, presione el botón 5 y se escribirá una 'Z'.
- ¿Y pulsar 'ENTER'? Bastará con pulsar el botón 5 la primera vez, ya que no esperará a una segunda pulsación y escribirá el código ASCII para 'ENTER'
