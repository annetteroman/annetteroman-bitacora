Gestión de Proyecto (Bitácora digital)

April 1, 2022 Annette R.

HEMOS VISTO:

- Descargamos el Visual Code Studio

- Elegimos una plantilla HTML y editamos

AULA VIRTUAL:

1. BITACORA: Cuaderno digital/escrito donde se lleva un registro de diversas acciones, donde se explica el proceso de investigacion o acciones.

Caracteristicas:

- Blog: Sitio web, el ultimo articulo es el mas reciente

- Web: "World wide web" es un navegador que se comunica con diferentes servidores.

- Redes sociales: Sitios y aplicaiones donde se interactua y comparte globalmente.

- Contenido Audiovisual: Producto con sucesion de imagenes o audios que es transmitido incluye cinematografia u otros

- Blog: Sitio web

2. REPORTE DE AVANCE: Informe academico que recoje los resultados de la información.

3. DOCUMENTACION: Procesamiento de información.

4. NOTA DE CLASE: Resumen de los temas desarrollados en clase

5. DIFICULTAD:

- Obstaculo: Aquello que dificultad el cumpliento

- Dificultad: Incovenientes o barreras que superar para lograr los objetivos

- Cometario: Apreciacion sobre un analisis

- Conclusión: Sintesis de puntos mas resaltantes.
